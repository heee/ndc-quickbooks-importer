How to import into QuickBooks Desktop 2017
  1. Back up your company file first — File → Back Up Company → Create Local Backup
  2. Go to File → Utilities → Import → IIF Files
  3. Browse to the downloaded .iif file and click Open
  4. QuickBooks will confirm: "Your data has been imported" — click OK
  5. Review the imported transactions in the relevant account register to verify
